## 💡 Sobre o Projeto
Este repositório documenta a transição da teoria para a prática no desenvolvimento de um sistema de alarme inteligente, integrando modelos avançados de detecção de objetos, como o YOLOv8n, para garantir precisão, configurabilidade de níveis de alerta e escalabilidade para futuras análises de comportamento.

## 🗂️ Estrutura do Repositório

O projeto é estruturado para guiar o leitor desde o entendimento dos conceitos matemáticos e teóricos até a execução prática da solução:

> 📁 **1_TEORIA**
> * [contexto_teorico.md](1_TEORIA/contexto_teorico.md): Análise técnica sobre como Redes Neurais Convolucionais transformam frames de vídeo em eventos de segurança.

> 📁 **2_PRÁTICA**
> * [0_informacoes_projeto.md](2_PRATICA/0_informacoes_projeto.md): Guia de arquitetura, configurações do sistema e explicação da lógica dos "Alertas Inteligentes" em dois níveis.
> * [1_analise_exploratoria.ipynb](2_PRATICA/1_analise_exploratoria.ipynb): Notebook dedicado ao entendimento do dataset e comportamento dos dados.
> * [2_implementacao_projeto.ipynb](2_PRATICA/2_implementacao_projeto.ipynb): Notebook principal com a implementação do modelo, inferência em tempo real e integração do disparo de alertas.
